# Hannah Lucas Portfolio v2 — Anti-Gravity Executive Brief (Runway Studio Edition)

## Goal
Transform the current site into a **cinematic interactive journey** + **Verification archive** + **speaker/press conversion machine**,
while giving Hannah **full no‑code control** to update everything forever (drag‑and‑drop, click‑to‑edit, upload media, publish).

## Aesthetic
High‑fashion futuristic + bedazzled innovation + early‑2010s runway‑opener nostalgia (flashbulb + glossy‑mag energy) + documentary montage vibe (personal artifacts),
but **business and founder‑coded**.

## Core UX
- **Journey mode:** 7 scrollytelling chapters with autoplay‑muted media and 4 “Impact Spike” moments.
- **Verification mode:** filterable archive of every hit, award, speech, project, partnership, policy mention.
- **Runway Studio admin:** a visual editor like Wix/Canva (click to edit, drag modules, replace media, publish).

## Non‑negotiables
- Replace “receipts/proof” language with **VERIFICATION / VALIDATION** everywhere.
- Add custom cursor using the provided heart monogram image; keep **cursor color change** feature without ever removing the cursor.
- Autoplay video on scroll **muted**. Audio only after user toggles **Sound: ON**.

## Backend requirements (no code for Hannah)
- Visual editor with module library, reorder via drag‑and‑drop, inline editing
- Asset library for any media type + embeds
- Chapter builder for Journey
- Verification builder for archive + bulk import from spreadsheets
- Theme & Motion panel (palette, fonts, sparkle density, flashbulb intensity, Runway vs Calm mode)
- Cursor settings panel (enable, size, tint color)

## Deliverables
- /studio (Runway Studio) admin + roles
- Content types: Chapters, Verification Entries, Speaking, Press Kit, Assets, Theme/Motion, Site Settings
- Autoplay media system (IntersectionObserver)
- Search + filters for Verification archive
- Cursor system (image preserved, tint overlay)
